# Martial Arts Instructor Landing Page

A modern, responsive landing page for a martial arts instructor specializing in Brazilian Jiu-Jitsu and MMA. The site features a dark theme with blue and slate accents, following a mobile-first approach with clean code organization.

## Features

- Floating navbar with smooth scrolling
- Hero section with impactful background image
- Welcome video section
- Programs section with accordion and image galleries
- Certifications gallery with modal image viewing
- Call-to-action section with statistics and images
- Comprehensive footer

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui components

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Project Structure

- `/src/app`: Next.js app router files
- `/src/components`: React components
  - `/ui`: shadcn UI components
  - Other components organized by feature
- `/public`: Static assets

## Design Principles

- Mobile-first responsive design
- Dark theme with blue and slate color palette
- Clean component separation
- Client components for interactive elements
- Accessibility considerations
